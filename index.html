At Studio Alonzo I enjoy crafting things in a more hands on manner using old world techniques and methods. Crafting wood so that it's natural radiance shines through so that it can be enjoyed for many years. Whethere it is a keepsake box, table or anything in between it can be made. I make my own shellac and use a food-safe contact polish to create that age old look with a clean and clean shine.
